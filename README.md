# 🌱 Farmer Comrade — AI Crop Doctor
### వివసాయి తోజన్ | किसान साथी | விவசாயி தோழன்

> An AI-powered agricultural assistant that helps farmers identify crop diseases, 
> get treatment advice, and hear results in their own language.

---

## 🎯 Problem We Solve
Every year, Indian farmers lose **₹50,000 crores** due to crop diseases.
Most farmers cannot afford agricultural experts or don't have access to them.
**Farmer Comrade** puts an expert AI in every farmer's pocket — for free.

---

## ✨ Features
- 📸 **Photo Analysis** — Upload any crop photo and get instant diagnosis
- 🔍 **Disease Detection** — AI identifies diseases, pests, and health issues
- 💊 **Treatment Advice** — Cheapest and easiest remedies suggested
- 🌦️ **Weather Warning** — Know which weather makes disease worse
- 🌾 **Soil Tips** — Quick soil health recommendations
- ⚡ **Urgency Level** — Know how fast to act
- 🔊 **Voice Output** — Hear results in your own language
- 🌍 **8 Languages** — English, Tamil, Hindi, Telugu, Kannada, Malayalam, Marathi, Bengali

---

## 🛠️ Tech Stack
| Technology | Purpose |
|-----------|---------|
| Python + Flask | Backend server |
| Groq AI (Llama 4) | Crop analysis AI |
| gTTS | Text to speech |
| HTML + CSS + JS | Beautiful web UI |

---

## 🚀 How to Run

1. Clone the project
2. Install dependencies:
\`\`\`bash
pip install flask groq gtts
brew install mpg123
\`\`\`
3. Add your Groq API key in main.py
4. Run:
\`\`\`bash
python3 main.py
\`\`\`
5. Open http://127.0.0.1:5000
